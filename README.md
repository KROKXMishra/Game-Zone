# 🌌 Game Zone: The Ultimate Gaming Hub

Welcome to **Game Zone**, a high-performance, neon-themed gaming hub built with HTML5, CSS3, and Vanilla JavaScript. This collection features five distinct strategy and logic games, all unified under a sleek glassmorphism aesthetic.

## 🕹️ Select Arena

### 1. ♟️ Chess Arena
A fully functional chess engine with legal move highlighting and turn management.
* **Features:** Legal move validation, neon white/cyan path glows.

### 2. 🔢 2048 Arena
The classic mathematical puzzle. Slide tiles to reach the legendary 2048 block.
* **Features:** High-score persistence via `localStorage`, smooth tile merging animations.

### 3. 🧩 Sudoku Solver
A complex grid-based tool that can solve any valid Sudoku puzzle using backtracking algorithms.
* **Features:** 3x3 sub-grid visual separation, neon input highlighting.

### 4. ⭕ Tic-Tac-Toe
The classic strategy duel between X and O.
* **Features:** Win pattern detection, draw logic, and player-specific neon glows.

### 5. ✊ Rock Paper Scissors
A reflex-based game against a randomized computer opponent.
* **Features:** Dynamic messaging, score tracking, and glassmorphism UI.

---

## 🛠️ Built With

* **HTML5:** Semantic structure and game layouts.
* **CSS3:** Advanced Flexbox/Grid systems, glassmorphism, and radial neon gradients.
* **JavaScript:** Backtracking algorithms, move validation, and state management.

## 🚀 How to Play

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/KROKXMishra/Game-Zone.git](https://github.com/KROKXMishra/Game-Zone.git)
    ```
2.  **Launch the Hub:**
    Open the root `index.html` in any modern web browser.
3.  **Navigate:**
    Use the main menu to select an arena. Use the "Back to Hub" link in each game to return to the main menu.
